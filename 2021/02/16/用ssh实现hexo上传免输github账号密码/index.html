我们需要寻找一对 id_rsa 或 id_dsa 命名的文件，其中一个带 .pub 扩展名。
其中 .pub 文件是你的公钥，另一个则是私钥。
如果没有找不到这样的文件（ 或者根本就没有 .ssh 目录 ），我们可以通过 ssh-keygen 程序来创建它们。

首先 ssh-keygen 会确认密钥的存储位置和文件名（默认是 .ssh/id_rsa）,然后他会要求你输入两次密钥口令，留空即可。
所以一般选用默认，全部回车即可。
